## 👁‍🗨Funcionalidades do projeto

Projeto dividido em 3 partes contendo as seguintes informações
### 1º parte
- Cabeçalho
    - Logo
    - Navegação
        - Quem somos
        - Onde estamos
        - Contato
### 2º parte
- Conteúdo principal
    - Painel
        - painel com imagem
    - Seção com texto 1/2
        - Quem somos
        - Lorem Ipsum
    - Ladrilhos
        - Ladrilhos de imagens
    - Seção com texto 2/2
        - Onde estamos
        - Endereço
    - Contato
        - Contato
        - Dados para entrar em contato com o estabelecimento
### 3º parte
- Rodapé
    - Logo
    - Navegação
        - Quem somos
        - Onde estamos
        - Contato
    - Conteúdo de baixo
        - Redes sociais
        - Copyright
